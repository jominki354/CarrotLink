## 기능

### 홈 화면
- 현재 설치된 브랜치, 버전 정보 확인
- 기기 고유 ID (Dongle ID, Serial) 확인
- Google Drive 연동 시 주행 영상 및 경로 확인
- 백업 상태 확인

### Git 관리
- 최신 버전으로 업데이트 (git pull)
- 다른 브랜치로 변경
- 문제 발생 시 원래 상태로 되돌리기

### 시스템 제어
- 소프트 재시작 (화면만 다시 시작)
- 기기 재부팅
- 재빌드 (소스 코드 다시 컴파일)
- 학습 데이터 초기화
- 카메라 보정 초기화
- 녹화 영상 삭제

### 터미널
- 직접 명령어 입력 가능
- 가상 키보드 지원
- 자주 쓰는 명령어 저장 (매크로)

### 파일 탐색기
- 기기 내 파일 확인 및 관리
- 파일 업로드/다운로드
- 텍스트 파일 직접 수정
- 자주 가는 폴더 즐겨찾기

### 자동 백업
- 설정값 변경 시 자동으로 백업
- Google Drive에 저장
- 백업 주기 설정 가능
- 이전 백업으로 복원 가능

### 설정
- 연결 정보 관리
- GitHub 계정 연동
- 백업 주기 설정
- 녹화 공유
- 업데이트 채널 선택 (Stable/Dev)
